dboydgit | Free Code Camp | url Shortener Microservice

How to use

Pass a url to "https://dbsurl.glitch.me/create" as a parameter, the service will return a JSON string object with a shortened url which will redirect to the provided address.

Examples

input: https://dbsurl.glitch.me/create/https://www.google.com

output: {original_url: "https://dbsurl.glitch.me/create/https://www.google.com", short_url: "https://dbsurl.glitch.me/1"}

Usage

"https://dbsurl.glitch.me/1" will now redirect to "https://www.google.com"
